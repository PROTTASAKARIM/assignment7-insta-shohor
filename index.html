What is the difference between local storage and session storage?

local storage saves all data even after closing the window.If after closing the window we reopens the window we will see the data we saved before is stored in the local store. We can save item in local store by localStorage.setItem(key, value).

Unlike local store session stores the data temporarily. We can save all and any kind of data we like in the session store but after closing the window if we reopens the window we will not find those data anymore. we can save items in session store by sessionStorage.setItem(key, value).

local store and session store has its own usefullness. If we want to save data permanently local store is need as sessionstore does not stores the data permanently. And if we want to test some data and note save the data in the server permanently session store is better,
